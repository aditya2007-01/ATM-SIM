# 🏧 C++ ATM Machine Simulator

A classic, interactive command-line banking simulator written in C++. This project demonstrates foundational programming concepts like continuous menu loops, conditional branching, and state management in a clean, terminal-based interface.

## ✨ Features

* **Interactive Menu Loop:** Built with a `do-while` loop so the menu continuously reappears until the user explicitly chooses to exit.
* **Live Balance Tracking:** Maintains and updates account balance dynamically across multiple deposits and withdrawals in a single session.
* **Overdraft Protection:** Includes logic validation to prevent users from withdrawing more money than their current available balance.
* **Clean Console I/O:** Well-formatted terminal output with visual separators for a smooth user experience.

## 🛠️ Concepts & Technologies Used

* **Language:** C++
* **Core Concepts:**
  * Variables & Data Types (`int`)
  * Control Flow (`if`, `else if`, `else`)
  * Iteration & Loops (`do-while`)
  * Standard Input/Output (`std::cin`, `std::cout`)
